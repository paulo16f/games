# City Builder

Planned adaptation of the Lumberjack on-chain example as a no-deploy browser game.

## Concept

- Place buildings to generate resources over time
- Gacha pulls for blueprints and upgrades
- Server-side resource ticks, wallet connect required
- Based on the Lumberjack Anchor program (reference only, not deployed)
